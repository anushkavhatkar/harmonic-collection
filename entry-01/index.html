love letters to colour

by Anushka Vhatkar

entry 01: exposition

Colour has fascinated me for as long as I can remember. I didn't quite understand how it worked when I was much younger. It definitely felt like magic in a lot of ways. Having understood how it works now, it still feels like magic. My harmonic collection is an ode to everything that thrills me about colour. It is a journey into learning and designing information just as I learn to write and interpret code.

So let's start with something mildly boring:

  • If I told my younger self this wave completes 1 oscillation in 3 seconds, 7-year-old me might be able to tell its frequency is 0.33.

  • And if light is a wave, then it must have a frequency. It does.

    •       Turns out we already have a name for the frequency of light: color.

  • Color, really, seems to be nothing more than a measure of how quickly the light waves are waving
  • If our eyes were quick enough, we would've been able to see observe this periodic motion (just like we can see this wave!).
  • The frequency of the light we see is so high, we cannot possibly see it as a wave. It waves up and down more than 400 million million times a second.
  • But you're secretly a data-processor: you can tell by looking at the color of a light wave what its frequency is.
  • The lowest frequency light-wave we can see is red, and the highest frequency is violet.

    • And in between, all the other frequencies form a continuous band of color called the visible spectrum.

Say you have a yellow coloured object sitting on your desk. The sun emits all colors (all frequencies!) of light, so light of all colors is hitting this object. It appears yellow because it reflects yellow light more than any other colour. What about blue, purple, and red light waves?   They are absorbed, and their energy is converted to heat.

So in a way, the color we perceive an object to be is precisely the object it isn't.
